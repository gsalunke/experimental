Theme Configuration

Our library simplifies theming by focusing on brand colors. Just define your primary, secondary, and accent colors, and we'll handle all the variants and states.

{
  "palette": {
    "brand": {
      "primary": "#FF6B00",
      "secondary": "#FFB27F",
      "accent": "#FFE0CC"
    }
  },
  "brandComponents": {
    "borderRadius": 8,
    "spacing": 8
  }
}
